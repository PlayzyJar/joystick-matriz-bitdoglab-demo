# Joystick LED Controller  

✨ **Integração simples entre joystick e matriz de LEDs**  

## Tecnologias  
- **C** (Pico SDK)  
- **RP2040** (BitDogLab)  
- Matriz LED 5x5  

## Como Funciona  
- Direções do joystick → Notas musicais  
- Intensidade → Oitava alta/baixa  
- Centro → LED central
